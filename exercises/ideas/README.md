# Agent Skill Ideas

This directory contains structured descriptions of potential agent
skill ideas that could be implemented. Each idea has its own
subdirectory with a detailed description, evaluation approach, and
implementation pointers.

## Available Ideas

### 1. [PR Review: Prioritize Code Reviews](pr-review/README.md)
Automatically prioritize pull requests based on complexity, urgency,
and team capacity.

### 2. [Test Gap Finder: Find Missed Test Cases](test-gap-finder/README.md)  
Analyze code coverage and identify critical test gaps in your
codebase.

### 3. [Meeting Effectuator: Turn Meeting Notes into Tickets](meeting-effectuator/README.md)
Convert meeting notes and action items into structured tickets and
tasks.

### 4. [Documentation Drift Detector: Spot Outdated Documentation](documentation-drift-detector/README.md)
Detect documentation that has become outdated compared to the actual
codebase.

## Structure

Each idea follows this structure:

```
idea-name/
├── README.md              # Detailed idea description
├── eval-approach.md       # How to evaluate this skill
└── implementation.md      # Implementation pointers
```

## How to Use These Ideas

1. **Explore**: Read through the README files to understand each idea
2. **Evaluate**: Review the evaluation approach to understand how to test the skill
3. **Implement**: Use the implementation pointers as a starting guide
